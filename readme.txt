== Description ==

The Internet Explorer 8 Accelerator plugin enables you to post an entry with selected text of any web-page to your WordPress blog.

Many a times while browsing the internet we come across text which we would like to blog about. As a solution to this problem, Internet Explorer 8 includes a useful feature called Accelerators. These accelerators enable you to select text and post them immediately.

== Installation ==

1. Download the plugin file
2. Extract the archive contents to the wp-content/plugins folder i.e. wp-content/plugins/accelerator.php
3. Login to WordPress Administration Center
4. Activate the plugin
5. Goto http://anantgarg.com/wordpress-internet-explorer-8-accelerator/ to generate and add the WordPress Accelerator for your site.

For more details, please visit: http://anantgarg.com/wordpress-internet-explorer-8-accelerator/
